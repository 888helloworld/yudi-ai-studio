# 冰侠客小红书 AI 创作工具

面向小红书创作者、商家运营和内容团队的一站式创作工具。当前能力包括 AI 图片生成、爆款文案生成、文案改写、图文一体生成、积分体系、卡密兑换、模拟支付、个人中心和管理后台。

## 产品定位

帮助内容创作者把一个主题快速变成可发布的小红书素材，减少从想法、配图、文案到沉淀历史记录之间的重复劳动。

## 核心用户

- 个人创作者：需要高频产出笔记、标题、封面图。
- 小商家/探店运营：需要围绕商品、门店、活动生成种草内容。
- 内容团队：需要沉淀历史素材、管理积分、批量分发卡密。

## 当前功能

- 账号体系：注册、登录、JWT 鉴权、个人中心。
- 创作能力：图片生成、文案生成、文案改写、图文一体生成。
- 资产沉淀：用户历史记录、图片本地保存、积分流水。
- 商业化：积分套餐、卡密兑换、模拟支付订单。
- 管理后台：用户、历史、积分流水、卡密、支付订单。

## 本地运行

```bash
npm install
npm start
```

访问：

```text
http://localhost:3001
```

## 环境变量

复制 `.env.example` 为 `.env`，至少配置：

```env
ARK_API_KEY=
DEEPSEEK_API_KEY=
JWT_SECRET=
ADMIN_USERNAME=
ADMIN_PASSWORD=
ALLOWED_ORIGIN=http://localhost:3001
```

## 重要安全提醒

当前项目仍是轻量单体实现，正式上线前必须完成：

- 静态目录收敛到 `public/`，避免暴露源码和 `data.db`。
- 支付回调接入真实平台验签。
- 移除源码中的任何 API Key 兜底值并轮换已泄露密钥。
- 修复前端 `innerHTML` 带来的 XSS 风险。
- 完善上传文件头校验、下载超时和文件大小限制。
