## 版本规划

### V0.2 稳定可用

- 修复静态暴露和 API Key 泄露风险。
- 完成用户帮助、内容规范、隐私和条款页。
- 完成首页个人统计、历史统计和失败退款提示。
- 修复注册卡密重复使用问题。

### V0.3 可运营

- 增加生成质量反馈：好用、不准、违规、重试。
- 增加基础埋点和后台漏斗看板。
- 增加模板库：种草、探店、穿搭、美食、旅行、知识。
- 增加历史搜索、筛选、批量删除和导出。

### V1.0 可收费上线

- 接入真实支付平台和回调验签。
- 完成内容安全策略和敏感词拦截。
- 完成订单、发票、退款和客服处理流程。
- 增加生产部署说明、备份和监控。

## 验收口径

- 用户登录后可完成一次图片、文案、改写、图文创作。
- 每次成功创作后，积分、历史和顶部统计同步更新。
- 失败不错误扣分，退款流水可追溯。
- 普通用户无法访问管理员接口。
- 未登录用户不能调用生成接口。
- 生产环境不暴露源码、数据库、日志和测试文件。
