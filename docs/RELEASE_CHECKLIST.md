# 发布验收清单

## 功能验收

- 登录、注册、退出流程正常。
- 注册必须消耗有效邀请码或按配置关闭邀请码。
- 图片生成、文案生成、文案改写、图文一体均可成功生成。
- 积分扣减、失败退款、流水记录一致。
- 历史记录可查看、删除、分页。
- 管理员可查看用户、充值、重置密码、生成卡密、查看订单。

## 安全验收

- 只暴露前端静态目录，不暴露项目根目录。
- `.env`、`data.db`、源码、日志、测试文件无法通过浏览器下载。
- 支付回调必须校验平台签名。
- API Key 不存在于源码中。
- 前端渲染用户输入和模型输出时不使用不可信 `innerHTML`。
- 上传文件做 MIME 和文件头双重校验。
- 生成接口有速率限制和积分校验。

## 数据验收

- 数据库有备份策略。
- 关键表有索引：用户、历史、积分流水、订单、卡密。
- 用户删除时关联数据处理符合产品策略。
- 统计口径和前端展示一致。

## 体验验收

- 未登录用户看到明确登录引导。
- 积分不足时给出充值或卡密兑换入口。
- 生成失败时展示失败原因或可重试提示。
- 移动端按钮、表格、弹窗不溢出。

## 运营验收

- 帮助中心可解释积分、失败退款、卡密、支付状态。
- 内容规范页可解释禁止生成的内容类型。
- 隐私政策和服务条款有页面入口。
- 管理员知道如何处理用户充值、卡密、密码重置和投诉。
